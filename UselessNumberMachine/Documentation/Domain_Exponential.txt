Square Root Sum*
---------------
input:	list of numbers DS
output:	square root of the sum
psuedocode:
	find sum
	return sqrt sum

Sum Square Root*
---------------
input:	list of numbers DS	
output:	sum of square roots
psuedocode:
	while there are still numbers left
		sum += sqrt number
	return sum

Square Root Difference*
----------------------

output:	square root of differences
psuedocode:

Difference Square Root*
----------------------
input:	list of numbers DS	
output:	difference of square roots
psuedocode:
	while there are still numbers left
